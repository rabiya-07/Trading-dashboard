# Trading Analytics Dashboard 📈

An interactive Machine Learning and Data Analytics dashboard built with Python and Streamlit to analyze trader performance and Bitcoin market sentiment.

## 🚀 Features

* Market Sentiment Analysis (Fear vs Greed)
* Historical Trading Data Analysis
* Interactive Visualizations
* Machine Learning Integration
* Real-time Dashboard Interface

## 🛠 Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit
* Matplotlib
* Plotly

## 📂 Project Structure

```bash
project/
│── app.py
│── requirements.txt
│── README.md
│── dataset.zip
```

## ▶️ Run Locally

Clone the repository:

```bash
git clone https://github.com/rabiya-07/Trading-dashboard/tree/main
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit app:

```bash
streamlit run app.py
```
