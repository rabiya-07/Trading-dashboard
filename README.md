# 📊 Crypto Trading Sentiment Dashboard

An interactive Machine Learning dashboard for analyzing cryptocurrency trading performance and Bitcoin market sentiment.

Built using Python, Streamlit, Pandas, and Scikit-learn.

---

# 🚀 Features

* 📈 Trading performance analytics
* 🧠 Fear vs Greed sentiment analysis
* 🤖 Machine Learning profit prediction
* 📊 Interactive visual dashboard
* 📉 Daily PnL trend analysis
* 🪙 Coin-level insights
* 👤 Account-level performance tracking

---

# 🛠 Tech Stack

* Python
* Streamlit
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

# ▶️ Run Locally

## 1. Clone Repository

```bash
git clone https://github.com/rabiya-07/Trading-dashboard
```

## 2. Open Project Folder

```bash
cd project-folder
```

## 3. Install Dependencies

```bash
pip install -r requirements.txt
```

## 4. Run Streamlit App

```bash
streamlit run trading.py
```

# 🧠 Machine Learning

The project includes a Logistic Regression model to predict profitable vs non-profitable trades using:

* Trade Size
* Market Sentiment
* Rolling Sentiment Features

---

# 📌 Key Insights

* Trading sentiment impacts profitability
* Larger trade sizes may increase risk exposure
* Daily PnL trends help identify strategy stability
* ML can assist in predicting trade outcomes
